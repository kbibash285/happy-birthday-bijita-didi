# 🎂 Happy Birthday Bijita Didi ❤️

A special birthday webpage created with love for Bijita Didi!

## 🌸 Features

✨ **Beautiful Animated Design** - Floating flower animations and smooth scrolling
🎵 **Birthday Music** - Play your favorite birthday song
💌 **Heartfelt Message** - A personal message of gratitude and love
📸 **Photo Gallery** - Display beautiful memories with lightbox view
🎉 **Birthday Wishes** - Special birthday messages and blessings

## 📋 How to Use

1. **Upload Your Photos** - Add 4 photos to the repository:
   - `photo1.jpg` - Birthday celebration
   - `photo2.jpg` - Wedding photo
   - `photo3.jpg` - Airport send-off
   - `photo4.jpg` - Family portrait

2. **Visit the Website** - Open in your browser:
   🌐 https://kbibash285.github.io/happy-birthday-bijita-didi/

3. **Share the Link** - Send the URL to Bijita Didi to view the birthday surprise!

## 🎨 Customization

You can edit `index.html` to:
- Change the message
- Update the music link
- Modify colors and fonts
- Add more photos to the gallery

## 📸 Photo Requirements

- Format: `.jpg` or `.png`
- Size: Recommended 1MB or less for faster loading
- Dimensions: Square images work best (e.g., 1080x1080px)

## 🎵 Music

The birthday song is linked from YouTube. Make sure to add the music file or update the link in `index.html`.
